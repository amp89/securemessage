What is this?
This is a messaging app that allows users to communicate using encrypted messages. It is written using Python/Django (I still store the data in plaintext as well, for testing purposes).
The app works by creating an RSA key for each user when they sign up. When they sign in, the password is hashed and stored as a session variable. This hashed password is used to decrypt their RSA key. When sending a message, it is encrypted with the recipient's public key.

Purpose
I created this application to gain hands on experience with Django and further my knowledge of Python (I have done mostly Java for my own projects up until this point).
I have been playing around with Django for a little while now. It seems to be a very flexible framework with a lot of cool things built in that can reduce time spent rewriting code that has been written several times. I like how it lets the developer focus on the app that they want to make, instead of creating the same boiler plate code that is used in almost every project. For example, instead of writing code to ensure the user is logged in prior to performing certain actions, I can just make my class based views extend the "LoginRequiredMixin", and it is already done for me. I also really like the Models / ORM, and the forms are usefull for quick functionality.
